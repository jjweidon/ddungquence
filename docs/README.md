# 뚱퀀스(DdungQuence) 구현 문서 모음

> 스택: Next.js(App Router) + TypeScript + TailwindCSS + Firebase(무료 구간 최대 활용) + Vercel

## 목적
- Cursor 기반 바이브코딩을 빠르게 진행할 수 있도록 **요구사항/설계/컨벤션/로드맵**을 사전 정리한다.
- 자체 서버 없이 Firebase(BaaS)로 **턴 기반 실시간 동기화**를 구현한다.
