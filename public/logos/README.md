# DdungQuence(뚱퀀스) 로고 시안 패키지

본 ZIP은 **컨셉 검증/초기 개발용** 로고 시안입니다.  
(최종 출시/상용 배포 전에는 상표/유사 로고 리스크 체크를 별도로 권장)

## 폴더 구조
- `01-marks/` : 심볼(마크) 시안
- `02-lockups/` : 심볼 + 워드마크(한글/영문) 조합 시안
- `03-app-icons/` : 앱 아이콘 시안(1024x1024, 라운드 적용)
- `04-previews/` : 한 장 미리보기

## 추천 1순위(현재 기준)
- 심볼(마크): `mark02_slash5_ridges_*`
  - 포커칩 감성(트럼프 카드 룩) + **5라인(승리 라인)**이 가장 직관적으로 전달됨
- 앱 아이콘: `appicon01_primary_charcoal.png`
  - 다크 톤에서 대비가 좋고, iOS/Android 모두에서 형태가 안정적

## 마크 네이밍 규칙
- `slash5` : 대각선 방향 5줄
- `row5` : 가로 방향 5줄
- `dashed` : 점선 스타일
- `pips5` : 5개의 칩/핍(pip)을 라인으로 배치

## 컬러 컨셉(기준 팔레트)
- Black: #0B0B0F
- Charcoal: #1A1D24 / #111318
- White: #F8FAFC
- Red: #D61F2C

## 사용 가이드(요약)
- 작은 사이즈(<= 24px)에서는 `mark05_pips5` 처럼 디테일이 많은 시안은 뭉개질 수 있음
- Favicon(16~32px)에는 `mark01_slash5_flat` 또는 `mark03_row5`처럼 단순한 형태가 유리
